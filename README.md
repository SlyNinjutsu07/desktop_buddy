This project is just me exploring basic File I/O management using Java. 

As of right now, I'm looking to make this project feature capabilities of detecting
and editing markdown files and folders. 

#### Roadmap so far:

- [X] Added a sprite icon
- [X] Adding a note manager window with settings and add buttons
- [X] Writing/reading settings data to a local json on your device
- [X] Viewing folders and notes on the note manager window
- [ ] Deleting and adding notes and folders
- [ ] Editing markdown files in a note writer window
- [ ] Adding bolding, highlighting, italicizing, and additional properties to be viewable within the editor
- [ ] Sprite animations (much later)

**This project will be my first version of building a desktop buddy. I plan on 
expanding upon this project so it works like a fully-fledged desktop 
assistant in the future.**

This project is also inspired by the **heyclicky** project made by farzaa.
Link to his [project](https://github.com/farzaa/clicky)

#### Future features (hopefully):
- [ ] AI assistant for personal storytelling and writing
- [ ] Spotify music list manager (or play music itself)
- [ ] Help organize files on your system like Claude CoWork
- [ ] Customizable sprite features
- [ ] Virtual assistant for navigating tasks on screen (like Clicky)
